Beschreibung:
RAWr ist eine für das Wahlpflicht Modul „Mobile Systeme“ entwickelte Zeichenreferenz-App für Android. Die App soll Künstlern die Organisation von Zeichenreferenzen erleichtern, die für die Kreation von neuen Kunstwerken nötig sind.

Umsetzung mit:
Visual Studio Code, Expo CLI, React Native, JavaScript, Node.js
