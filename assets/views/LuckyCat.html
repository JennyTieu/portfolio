Beschreibung:
Lucky Cat ist eine Webseite für Figuren Sammler und wurde von verschiedenen Online-Shops, sowie Preisvergleich-Webseiten inspiriert. Mit Funktionen, wie das Abspeichern von Lieblingsfiguren bzw. Must-Haves, das Bewerten von Figuren und Preisvergleiche bei ausgewählten Shops, kann die Kaufentscheidung für Figuren-Liebhaber erleichtert werden.


Umsetzung mit:
Visual Studio Code, Javascript, HTML, CSS und Node.js
