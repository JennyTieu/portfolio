Beschreibung:
Frogger ist eine auf Social-Media basierte App für Android. Dabei haben wir uns von derzeit viel genutzten Apps, wie Twitter und Facebook inspirieren lassen, um eine App mit frischem Konzept zu erstellen die, die gefragtesten Funktionen durchführen kann und für die neuere Generation attraktiv scheint.


Umsetzung mit:
Visual Studio Coder, Expo CLI, React Native, JavaScript, Node.js
